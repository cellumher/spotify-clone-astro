# Spotyfy Clone

Clon de Spotify desarrollado con Astro 4 y React. Al utilizar la API View Transitions, se recomienda utilizar un [navegador compatible](https://caniuse.com/?search=View%20Transition%20API) (Chrome, Edge, Opera)
